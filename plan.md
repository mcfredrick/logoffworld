1. Executive Summary
Log Off is a multi-modal digital ecosystem designed to counteract the isolation and addiction inherent in modern algorithmic social media.
Core Pillars

The Daily Census (MVP): A minimalist, ephemeral daily site at logoff.world. Users answer one daily question with two empowering choices. Features a "Live Pulse" animation showing real-time global participation. Funded by donations (80% to charity).
The Daily Audio (Companion): A 60-90 second AI-generated podcast episode released daily, featuring the prompt, a reflection, and a calming soundscape.
The Aggregate Discovery App (Future): A privacy-first content platform curating the "long tail" of the internet with a "Horizon Breaker" mechanic.
Real Human Thoughts (Future): A sister site for crowd-sourced philosophical insights.

Core Philosophy

Anti-Addiction: No streaks, no scores, no infinite scroll.
Radical Transparency: Users know exactly why they see content.
Shared Humanity: We are a collective experiencing the world together.
Virtuous Economics: Revenue supports the mission and charity, not shareholder value.


2. Product Architecture & Features
2.1. Project A: The Daily Census (MVP)
URL: https://logoff.world
Type: Single Page Application (SPA) / Static Site.
Core Features

The Daily Prompt:

One sentence, changing at midnight UTC.
Example: "Go outside and consider a cloud."


The Dual Choice:

Option A (Connection): "I found my sky today." (Empathetic, nature-focused).
Option B (Rebellion): "I marched to my own drum beat." (Autonomous, individual-focused).
Logic: Both options are valid. No "wrong" answer.


The Live Pulse (Real-Time Animation):

Trigger: When any user globally casts a vote, a floating emoji (☁️ or 🥁) animates on all currently open pages.
Effect: The main counter increments with a "pop" animation.
Throttling: If >5 votes occur in 1 second, show a single "wave" effect to prevent visual chaos.


The Ghost Counter:

Live, aggregated stats (e.g., "896 found their sky").
Ephemeral: All data resets to zero at midnight UTC. No history is stored.


Community Input:

Form for users to submit future prompt ideas.
Vetting: Manual review by founder before inclusion in the rotation queue.


Donation & Charity:

Split: 80% Charity, 15% Ops, 5% Founder.
Monthly Cause: Rotating charity (e.g., Jan: Mental Health).
Transparency: Dashboard showing funds raised/donated.
Donor Perks: Ad-free (if applicable), "Patron" badge.



Tech Stack

Frontend: HTML5, CSS3, Vanilla JavaScript (No heavy frameworks).
Hosting: Netlify or GitHub Pages.
Backend (API): Serverless Functions (Netlify Functions/Vercel Functions) or lightweight Node.js server.
Database: Redis (for atomic counters and ephemeral data).
Real-Time: WebSocket (Socket.io) or Server-Sent Events (SSE) via Supabase Realtime or Ably.
Payments: Ko-fi (one-time and recurring memberships).
Analytics: Plausible Analytics (Privacy-first, no cookies).


2.2. Project B: The Daily Audio (Podcast)
Type: Automated Audio Generation Pipeline.
Core Features

Content:

Intro: "Welcome to Log Off. Today's prompt is..."
The Prompt: Read clearly.
Reflection: 20-second soothing reflection generated by LLM based on the prompt.
Outro: "Join [Count] others who paused today. Be human."


Audio Design:

Voice: High-quality TTS (e.g., ElevenLabs).
Soundscape: Layered ambient/nature sounds (rain, wind, fire).


Distribution:

RSS Feed: Standard podcast feed for Apple/Spotify.
Web Player: Embedded on logoff.world.


Automation:

Nightly cron job generates MP3, updates RSS, publishes.



Tech Stack

Generation: Python script (calls TTS API + FFmpeg for mixing).
Storage: AWS S3 or Cloudflare R2.
RSS: Static XML file hosted on Netlify/GitHub Pages.


2.3. Project C: The Aggregate Discovery App (Future)
Type: Content Discovery Engine.
Core Features

Interest Stack: User selects 3-5 broad categories (Science, Art, Food, etc.).
Horizon Breaker: Mandatory daily article from a dramatically different category with dynamic AI explanation.
Integration: Embeds the Daily Census prompt as a wellness nudge.
Monetization: Freemium ($5/mo), Affiliate, Sponsorships.


2.4. Project D: Real Human Thoughts (Future)
Type: Sister Site (realhumanthoughts.com).
Core Features

Content: Crowd-sourced "shower thoughts" and philosophical insights.
Vetting: Manual review for depth and toxicity.
Attribution: Credit to submitter.

5. Content & Community Guidelines
5.1. Phrase Libraries

Connection Phrases: Focus on empathy, nature, bonds. (e.g., "I made a connection," "I listened to the wind").
Rebel Phrases: Focus on autonomy, individuality. (e.g., "I marched to my own drum," "I chose a different quiet").
Generation: Rotated daily from a pre-generated list of 100+ phrases.

5.2. Podcast Guidelines

Tone: Calming, reflective, non-preachy.
Length: 60-90 seconds.
Music: Licensed/royalty-free ambient sounds.

5.3. Prompt Vetting

Criteria: Actionable, positive, non-toxic, non-political (unless universally human).
Process: Manual review by founder.

5.4. Privacy & Analytics (Plausible)

Philosophy: "We measure the movement, not the individual."
Tracked: Total visits, referral sources, device type, custom event vote_cast.
NOT Tracked: IP addresses, user agents, cookies, individual history.
Footer Link: "We use Plausible Analytics. No cookies. No tracking."
